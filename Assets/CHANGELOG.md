# 更新日志

* 重构了角色工具
    * 为角色引入别名与向量语义归类, 便于检索
    * 为角色引入触及计数与最近提及系统, 与遗忘曲线系统一同工作, 避免召回不重要人物
    * 引入了软归档系统, 人物在 15 轮内没有被提及则进入软归档状态, 更不容易被召回
* 修复了设定的模型提示词会导致缓存失效的问题
* 修复了 Anthropic 接口格式未正确传递缓存参数的问题
* 修复了场景总结未正确写入的问题
* 修复了进入新场景后, 旧场景未及时标记关闭的问题
* 修复了回滚与重写无法支持场景变更事件的问题
* 修复了单个对话可能有多个正在进行的场景的问题
* 修复了单轮对话未正确关联到场景的问题
* 重构了上下文管理, 采用固定长度场景总结, 以避免出现超长场景, 并提高了缓存命中率, 降低整体成本
* 修复了叙事生成代理在部分情况下会重复发出流式请求调用工具的问题
* 修复了修正功能会意外取到不同会话内容的问题
